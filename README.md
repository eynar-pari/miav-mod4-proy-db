El presente proyecto implementa un flujo ETL que permite la integración de datos desde una base de datos transaccional (OLTP) hacia un almacén de datos (Data Warehouse).

El proceso ETL extrae información relevante del sistema operacional, transforma los datos aplicando limpieza, cálculo de métricas y adaptación al modelo multidimensional, y finalmente carga la información en las tablas de hechos y dimensiones del Data Warehouse para su posterior análisis.
